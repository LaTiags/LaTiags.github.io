# Tiago Alves — Portfolio

Personal portfolio website built with HTML, CSS and JavaScript.
Live at **[latiags.github.io](https://latiags.github.io)**

## Pages

- `index.html` — main portfolio page
- `weather.html` — weather app with live API data

## Features

- Dark / light mode with localStorage
- Click counter on the header button
- Weather app — real-time temperature & wind speed for 11 cities
- External API demo (JSONPlaceholder)
- Responsive design — mobile friendly

## Skills showcased

HTML5 · CSS3 · JavaScript · Python · IoT · Mechanical Engineering

## APIs used

- [Open-Meteo](https://api.open-meteo.com) — weather data
- [JSONPlaceholder](https://jsonplaceholder.typicode.com) — demo user data

## Run locally

No build step needed, just open `index.html` in a browser.
